# Docker compose CRUD Mascotas personas

CRUD de personas y mascotas creado con el MERN stack, posteriormente fue dockerizado.

1.- docker build -t "react-app" ./client/
2.- docker build -t "api-server" ./server/
3.- docker-compose up
